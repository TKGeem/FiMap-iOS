# FiMap-iOS
> TKGeem's iOS map application. search some facility informations.

## Installation & Setup

In terminal:

```bash
cd FiMap/
pod install
FiMap.xcworkspace/
```
and bulid in Xcode.

## Usage example

Search the surrounding facilities information from the map and find the name by word search.
